# myblog
使用：
git clone https://guthub.com/heyouquan/myblog
npm install
npm run dev  // 预览
npm run build  // 构建



功能：
前端页面：
（1）登录注册
（2）博文展示
（3）评论、点赞、阅读数
（4）跳页
后台管理页面：
（1）增加博文
（2）删除博文
（3）查看博文
（4）修改博文
（5）管理评论、点赞、阅读数

接口文档：
（1）登录
username---String
password---String
checkstatus---on/off
（2）注册
username---String
password---String
repassword---String
